Axios merupakan sebuah "promise-based HTTP Client" yang bersifat "isomorphic" (bisa di run pada browser dan nodejs dengan codebase yang sama) untuk node.js beserta browsernya. "server-side" menggunakan "native node.js http module", sedangkan "client-side" (browser) menggunakan XMLHttpRequests. Beberapa fitur yang dimiliki seperti membuat XMLHttpRequests dari browser, membuat http request dari node.js, mendukung "Promise" API, dll.
